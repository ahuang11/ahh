sci.get_avg(data, axis=0, times_idc=None, lats_idc=None, lons_idc=None, lvls_idc=None, change_lvl_order=False)[source]¶

Finds the areal and/or time average and/or level, but first, data dimensions must be in these orders: lat, lon; time, lat, lon; time, level, lat, lon OR time, lat, lon, level (if change_lvl_order).

Parameters:
  • data – (np.ma.array) - input data
  • axis – (tuple) - axis to average over
  • times_idc – (np.array) - time indices
  • lats_idc – (np.array) - latitude indices
  • lons_idc – (np.array) - longitude indices
  • lvls_idc – (float) - level indices
  • change_lvl_order – (boolean) - changes order of level dimension
Return avg:

(np.ma.array) - average over given parameters
