pre.concat_nc(glob_str, out_fi=None, in_dir='./', out_dir='./', rec_dim=None)[source]¶

Wrapper of NCO’s ncrcat and optional ncks; concatenates a list of netCDF files given a glob_str (i.e. ‘THETA.1440x720x50.2010*.nc’) and outputs concatenated file as the given out_fi. If there’s an error complaining that there’s no record dimension, can give the dimension name to concatenate across under rec_dim.

Parameters:
  • glob_str – (str) - the naming pattern of the files
  • out_fi – (str) - name of concatenated file
  • in_dir – (str) - directory of input files
  • out_dir – (str) - directory of output file
  • rec_dim – (str) - name of dimension to act as record dimension
Return out_path:
 

(str) - path to final concatenated file
