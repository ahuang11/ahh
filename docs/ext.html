ext.ahh(variable=None, n='ahh', ignore=None, col=None, time=0, level=0, center=0, offset=0, threshold=15, precision=2, edgeitems=5, suppress=True, snippet=True, fillval_high=99999.0, fillval_low=-99999.0, quiet=False, stop=False, **named_vars)[source]¶

Wrapper of one_ahh(); explores a variable.

Parameters:
  • variable – (array) - variable to be evaluated
  • n – (str) - name of variable
  • ignore – (scalar/str) - a value/str to ignore
  • col – (str) - if dataframe, name of column
  • time – (integer) - index of time to print in snippet
  • level – (integer) - index of time to print in snippet
  • center – (int) - if not 0, the number*2 to print around the center
  • offset – (int) - count of indices offset from the center
  • threshold – (int) - count before abbrieviating the print
  • precision – (int) - number of decimal places
  • edgeitems – (int) - how many numbers to print on the edge
  • suppress – (boolean) - whether to suppress scientific notation
  • snippet – (boolean) - whether to exclude snippet of values
  • fillval_high – (float) - anything equal/greater than fill value will not be included in the max
  • fillval_low – (float) - anything equal/less than fill value will not be included in the min
  • named_vars – (args) - named variables to be evaluated (fills in n)
  • quiet – (boolean) - whether to show snippet/center values
  • stop – (boolean) - whether to stop the ahh-ing in a loop
Return stop:

(boolean) - stop the ahh-ing in a loop
